API

zola_content_group

zola_content_group(name, deps, prefix, srcs, strip_prefix)

Specifies a group of files that should be mapped into the Zola tree.

ATTRIBUTES

NameDescriptionTypeMandatoryDefault
nameA unique name for this target.Namerequired
depsThe content that this group depends on.List of labelsoptional[]
prefixThe prefix to add to this content group when in the Zola tree.Stringoptional""
srcsThe srcs to add to this content group.List of labelsoptional[]
strip_prefixPrefix to strip from this targets path. To strip a path from the workspace root add '//' to the start of the path to strip otherwiseit is assumed that the strip should be relative to the current directory e.g. strip_prefix = '.', will just place the file directly in the Zola directory.Stringoptional""

zola_site

zola_site(name, config, content, sass, static, templates, theme, themes)

ATTRIBUTES

NameDescriptionTypeMandatoryDefault
nameA unique name for this target.Namerequired
configThe sites config.tomlLabelrequired
contentList of zola_content_groups.List of labelsoptional[]
sassSite sass style sheets.List of labelsoptional[]
staticStatic content.List of labelsoptional[]
templatesSite templates.List of labelsoptional[]
themeThe theme that is specified in the config.toml.Stringoptional""
themesA list of themes to add to this site. This is useful for themes that inherit from others. This should be used with the 'theme' attribute.List of labelsoptional[]
